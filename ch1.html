Briefly define:

  1. Computer Science: the study of the principles and use of computers
  2. algorithm: a set of instructions designed to complete a task
  3. sequential operation: the execution of operations one after each other
  4. condition operation: a logical operation used to test conditions
  5. iterative operation: an operation that needs multiple execution of instructions or processes
  6. stored program computers: a computer that stores memory to perform tasks intermittenly
  7. programmer: a person who gives specific instructions to computer/li>
  8. integrated circuit: chips made of semiconductor material used as a microprocessor, amplifier, or memory

Briefly describe these computer generations

  1. first generation computer: big heavy machines made out of vaccuum tubes, they had no operating system
  2. second generation computer: cheaper, faster, more reliable, and used magnetic cores and tape as memory and storage, also the start of using high level languages
